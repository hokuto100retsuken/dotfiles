指定されたテーマについて調査・ドキュメントを作成し、結果をクリップボードにコピーする。

## 手順
1. 指定されたテーマについてコードベースを調査する
2. 調査結果や設計書をMarkdown形式でまとめる
3. 完成したドキュメントをクリップボードにコピーする
4. 「クリップボードにコピーしました」と伝える

## 出力のポイント
- Markdown形式で整理する
- 簡潔に要点をまとめる
- コード例がある場合はコードブロックで囲む
- 不要な前置きや説明は入れない

## クリップボードへのコピー方法

生成した Markdown を標準入力から流し込む。ヒアドキュメントまたは一時ファイルでエスケープ事故を避ける。

```bash
# macOS
cat <<'EOF' | pbcopy
<ここに Markdown 本文>
EOF

# Linux
cat <<'EOF' | xclip -selection clipboard   # または xsel --clipboard
<ここに Markdown 本文>
EOF
```

$ARGUMENTS（対象テーマ。省略時はユーザーに確認する）
