# Gemini CLI configuration

Gemini CLI 関連の設定を管理します。

## Skills

`~/.gemini/skills` にシンボリックリンクされます。

### dotfiles-expert
このリポジトリの構造や規約（Fish プロンプト、Neovim 設定、カラーパレットなど）を Gemini CLI に理解させるためのスキルです。

### refactor-expert
コードの可読性や保守性を向上させるための専門スキルです。設定の分割や古い記法の更新など、リファクタリングを行う際に使用します。

### secret-guard
リポジトリ内に機密情報（APIキー、トークンなど）が混入していないかチェックするためのセキュリティスキルです。
